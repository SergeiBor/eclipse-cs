Eclipse Checkstyle Plug-in

Duplicate code detection
Duplicate code detection allows you to find code that has been generated by Copy/Paste programming. Checkstyle's implementation is StrictDuplicateCode, which means that it looks for duplicates without Java language knowledge.

To launch the analysis, right click on a workspace resource (project, folder, package or file) and go to "Checkstyle -> Search duplicated code" in the popup menu.

The result of the analysis will be displayed in a view where the duplicates are listed. If you double click on an item of the tree list, both of the two parts of code will open in Java editors.

Note : By default, the minimum amount of line to consider code as a duplicate is 20. This amount can be configured in the plugin preferences.
